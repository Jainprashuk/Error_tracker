# Bug Tracker SDK

A lightweight JavaScript SDK for automatically tracking frontend errors and API failures and sending them to your monitoring backend.

This SDK captures:

* JavaScript runtime errors
* Unhandled promise rejections
* Axios API failures
* Fetch API failures

Errors are automatically sent to your collector service.

---

# Installation

```bash
npm install bug-tracker-sdk
```

---

# Usage

```javascript
import { initBugTracker } from "bug-tracker-sdk";
import axios from "axios";

initBugTracker({
  project: "frontend-app",
  collectorUrl: "https://your-collector-url",
  axios
});
```

---

# What it captures

### 1. JavaScript Crashes

Example:

```javascript
const obj = null;
console.log(obj.name);
```

Automatically reported.

---

### 2. Axios API Failures

```javascript
axios.get("/api/payment");
```

Any failed request will be reported.

---

### 3. Fetch Failures

```javascript
fetch("/api/user");
```

Non-200 responses and network failures are tracked.

Example collector:
